Highlight reel builder

Trim the edges. Keep the best moments.

Clip Cutter is a lightweight desktop app for slicing recorded videos into clean highlight clips. Drag a file in, mark ranges on the timeline, and export instantly with ffmpeg stream copy.

View on GitHub Download
Windows · Works great for 30s–2m clips · Wails + Go



  1. Drag a recording into Clip Cutter.
  2. Mark highlight ranges on the timeline.
  3. Queue multiple clips for batch export.
  4. Export individual clips or all queued selections.
